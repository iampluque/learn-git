# Advice
- Rebase everyday
- Always Rebase before merging (Fix conflicts in your branch)
- Set your git global settings to ff-only to enforce Rebase behavior
  ```.gitconfig
  [merge]
    ff = only
  ```
- Work with Short-Lived branches
- Code Small Module
- Communicate
